# Viral Title Structure Analysis

Este projeto explora a estrutura invisível de títulos virais em plataformas como YouTube.

O objetivo é identificar padrões recorrentes de atenção utilizados em títulos que apresentam alta taxa de clique (CTR).

Eu estudo padrões estruturais de atenção em conteúdo digital.

No GitHub eu organizo análises de engenharia reversa de títulos, thumbnails e narrativa para identificar padrões recorrentes de retenção e clique.

A análise considera:

- gatilhos psicológicos
- arquitetura narrativa
- segmentação de nicho
- sinais de novidade
- promessas dramáticas

## Estrutura recorrente identificada

PERSONAGEM + ANO + PROMESSA DRAMÁTICA + CATEGORIA + FORMATO PREMIUM

ou de forma abstrata:

Entidade → Atualidade → Impacto → Nicho → Qualidade

Esses elementos atuam como gatilhos cognitivos que reduzem a fricção da decisão de clique.
